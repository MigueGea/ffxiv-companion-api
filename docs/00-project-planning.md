## Project Vision

FFXIV Companion API is a modern REST API built with Java and Spring Boot that provides access to Final Fantasy XIV character information.

The project is designed as a production-ready backend application and as a portfolio project to demonstrate modern Java development practices.

## Project Goals

The goal of this project is not only to build a functional API but also to serve as a portfolio project that reflects the architecture, code quality, and development practices expected from a modern Java Backend Developer.

## MVP

The first version of the API will include:

- Search characters by name
- Retrieve a character by ID

Additional features such as jobs, mounts, minions, achievements, and more will be added in future iterations.
